Getting Started
 
 
Implementation Steps

Login PERCEPTRON as a guest (without user account) or register for PERCETRON user account to login using an existing email account.

Step 1 – Basic Parameters
save_alt
  • Go to the ‘Protein Search Query’ in the menu on left and enter ‘Protein Search Title’ to start proteoform search
  • Browse and upload Data file(s). Upload one file (.mzXML/ .mzML/ .MGF/ .txt/) to perform single file search or add a zipped folder with multiple peak-lists (zipped .txt files) to search in batch mode
  • Browse for the folder containing database(s)
  • Select a Database to search from (e. g. Human)
  • Enter a False Discovery Rate (FDR) percentage to shortlist identified proteoforms at desired FDR cut off
  • Select the number of proteoforms identified to be received via email
    • Note: Users logging in via Guest Account can provide email address to receive results on Email ID upon search completion
 
 
 
 
Step 2 – Set Experimental Parameters
save_alt
  • Select the Mass Mode. MS data can only be provided in either m/z form with z = 1 (MH+) or neutral masses (M(Neutral))
  • Click ‘Filter Database using MS1’ to filter protein database on the basis of precursor ion mass (MS1)
  • Enter the desired ‘Intact Mass Tolerance’ and select its unit (Da or ppm)
  • Enter your desired ‘Peptide Tolerance’ and select its unit (Da or ppm)
  • Select ‘Tune Intact Protein Mass’ to allow for tuning of MS1 using MS2 data
  • Select the ‘slider value’ to provide a range for tuning MS1 while using ‘Tune Intact Protein Mass’ option
  • Select the fragmentation type from the drop-down menu
  • Choose the corresponding special ions for the type of fragmentation method selected (i. e. a’, b’, y’, z”, a*, b*, y*, z’ ions)
  • Provide the value of Neutral loss, if any
 
 
Step 3 – Set De Novo Sequencing Parameters
save_alt
  • Select ‘Enable PST Filtering’ to allow for peptide sequence tag (PST)-based search
  • Select your desired minimum PST tag length from the drop-down menu
    • Note: Input value for Minimum and Maximum Tag length should be different. Minimum Tag Length should be smaller than Maximum Tag Length.
  • Select your desired maximum PST tag length from the drop-down menu
  • Set the desired tolerance for each PST hop
  • Set the overall mass error tolerance for the entire PST
 
 
Step 4 – Set Protein Modifications Parameters
save_alt
  • Check ‘Handle Truncated Proteoforms’ to allow search for truncated proteoforms
  • Select ‘Terminal modifications’ to be included in the search
  • To allow search for unknown post-translational modifications (PTMs), search for Blind PTMs instead of variable or fixed modifications
  • Set the ‘PTM tolerance’
    • Note: If ‘Blind-PTM Search’ is selected, there is no need to set PTM tolerance.
  • Select ‘Fixed’ or ‘Variable’ post-translational modifications (PTMs) from the List of Modifications
  • Select the ‘Methionine Chemical Modifications’ from the drop-down menu
  • Select the ‘Cysteine Chemical Modifications’ from the drop-down menu
 
 
Step 5 – Set Scoring Components Weight
save_alt
  • Set the weights of the scoring components by moving the sliders
  • Submit the Protein Query Search by clicking on ‘Submit’
 
Step 6 – Proteoform Search Results
save_alt
  • Click on ‘Search Results & History’ to see the search results
