Getting Started
 
 
Implementation Steps
Step 1 – Basic Parameters
save_alt
  • Enter the ‘Protein Search Title’. Browse and upload Data file(s) to search in single or batch modes
  • Enter intact protein mass (MS1)
  • Browse for the folder containing database(s)
  • Select Database File (e.g. Human)
  • Select the number of candidate protein hits to be received in results
 
 
 
 
Step 2 – Set Experimental Parameters
save_alt
  • Select Mass Mode
  • Click ‘Filter Database using MS1’ to filter protein database on the basis of MS1 and ‘Handle Truncated Proteoforms’ to allow search for truncated proteoforms
  • Enter intact mass and peptide tolerances and select ‘Tune Intact Protein Mass’ to allow for tuning of MS1 using MS2 data
  • Select fragmentation type and corresponding special ions
  • Incorporate masses of neutral loss, if any
 
 
Step 3 – Set De Novo Sequencing Parameters
save_alt
  • Select ‘Enable PST Filtering’ to allow for peptide sequence tag-based search
  • Select for PST length and tolerance
 
 
Step 4 – Set Protein Modifications Parameters
save_alt
  • Select ‘Terminal modifications’ to be included in the search
  • Select ‘Fixed’ and ‘Variable’ post-translational modifications or click ‘Blind-PTM Search’ to allow search for unknown post-translational modifications
 
 
Step 5 – Set Scoring Components Weight
save_alt
  • Set ‘Scoring Components Weightage’ and click ‘Submit’
  • Check email for results or go to ‘Search Results & History’ page to view results
 
Step 6 – Summary and Detailed Results
save_alt
  • Go to ‘Search Results & History’ page to view summary results
  • Select the protein to see detailed results or check email for results
